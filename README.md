# news-app

## A news android app with the ability to display new popular news, save your reading list, and search for specific news on demand
![Preview](https://media1.giphy.com/media/ywuN2G3BgSIL0HS9Jk/giphy.gif)
## Technologies
 * Kotlin
 * MVVM
 * Navigation
 * Retrofit
 * Room
 * Recycler view
 * Data binding
 * Material components
 * Glide
